Required: HTML

All websites are written in HTML - Hypertext Markup Language. HTML is formatted using tags as shown in the window to the right. The windows shows the code that displays the title "This is a webpage". Tags are formatted using the < and > symbols, which let your browser know that it is being passed information. You can then put text inbetween two tags of the same kind, where the second begins with a / to show that it is closing the information, as shown here to display the title.

Highly Recommended: CSS

Where HTML decides what is on the page, CSS displays how. Cascading Style Sheets are used to describe colours, sizes, layouts, and various other options for how a webpage should be displayed. Modern browsers differ in what they are capable of doing on a webpage; for instance, Google Chrome, Mozilla Firefox and Apple's Safari provide a larger range of options for CSS than, say, Microsoft's Internet Explorer. Options could include fading elements in and out, rotating them, colouring them, and adding shadows, but a huge range is available and the list grows all the time.

Recommended: JavaScript

JavaScript is a programming langauge that opens ip a huge range of new options for your website. It could be that you want a window to pop up when someone hovers over a link, or collecting information from a user using a form. JavaScript has its limitations, but provides a powerful, customisable suite of options to add to your webpage.
